> Load antd to your create-react-app app via react-app-rewired.

## Install

```
$ npm install react-app-rewire-antd
```

## Usage

```js
const reactAppRewireAntd = require('react-app-rewire-antd');

reactAppRewireAntd('unicorns');
//=> 'unicorns & rainbows'
```
